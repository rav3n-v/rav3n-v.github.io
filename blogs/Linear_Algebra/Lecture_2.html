Introduction

In the last Lecture you have seen what are vectors and slight less formal definition of Vector Space. Keeping in mind \( \mathbb{R}^n \) as an example of Vector space with scalars from Field \( \mathbb{R}\). We will now formally define what is Field and a Vector Space

Definition of Field

A field is a set F equipped with two binary operations: addition and multiplication, satisfying the following properties for all elements a, b, c in F:

  • Associativity of addition and multiplication: a + (b + c) = (a + b) + c, and a · (b · c) = (a · b) · c.
  • Commutativity of addition and multiplication: a + b = b + a, and a · b = b · a.
  • Additive and multiplicative identity: There exist distinct elements 0 and 1 in F such that a + 0 = a and a · 1 = a.
  • Additive inverses: For every a in F, there exists an element in F, denoted −a, such that a + (−a) = 0.
  • Multiplicative inverses: For every a ≠ 0 in F, there exists an element in F, denoted a−1 or 1/a, such that a · a−1 = 1.
  • Distributivity: a · (b + c) = (a · b) + (a · c).

Note: Since addition and multiplication are defined as binary operations, the field is automatically closed under these two operations. Therefore, closure does not need to be stated separately.

Examples of Fields

So we know that \(\mathbb{R}\) is a field. But what about \(\mathbb{R}^2\)? If anyone asks you such a question, your reply should be "What is the definition of addition and multiplication for the field". Suppose they reply that addition and multiplication is defiend component-wise. Then \(\mathbb{R}^2\) is not a field.
Click to see why \( \mathbb{R}^2 \) is not a field

Assume addition and multiplication are defined component-wise:

(a,b) + (c,d) = (a+c, b+d) (a,b)(c,d) = (ac, bd)

The multiplicative identity is (1,1). Consider the nonzero element (1,0). If \( \mathbb{R}^2\) were a field, there must exist (a,b) such that: (1,0)(a,b) = (1,1) But (1,0)(a,b) = (a,0), which can never equal (1,1). Hence (1,0) has no multiplicative inverse.

Therefore, ℝ² is not a field.
